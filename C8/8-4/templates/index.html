1. 内联代码
html中<section>标签不是用来专门做容器的标签，在 <section>...</section> 可以来设置样式或脚本。
2.<kbd>...</kbd>是提示用户要输入的内容：
请在cmd命令行输入关机命令: shutdown -s
3.代码块放到<pre>...</pre>中 
01   from flask import Flask
02   from datetime import datetime
03   import config
04   from flask_sqlalchemy import SQLAlchemy
05   app = Flask(__name__)
06   app.config.from_object(config)
07   db = SQLAlchemy(app)#db = SQLAlchemy(app)需要放在config的后面，否则会有警告
08   db.init_app(app)
4.变量放到<var>...</var>中
y = mx + b
